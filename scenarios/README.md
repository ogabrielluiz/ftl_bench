# scenarios

Benchmark scenario definitions, each with a **pinned seed** and success/score criteria.

Two families:
- **Full run** — sector 1 → flagship; headline score (sectors cleared, flagship kills, final score).
- **Micro-encounters** — cheap, high-signal probes of specific skills: a single combat, an escape decision, a store-allocation puzzle, an event risk choice.

Each scenario records its reproducibility manifest: `{FTL version, Hyperspace build hash, mod bundle hash, Advanced Edition flag, seed}`.
